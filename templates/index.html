Set API keys in the shell before launching webui.py: SHODAN_API_KEY=… VT_API_KEY=… ABUSEIPDB_API_KEY=… python3 webui.py

Sherlock-style sweep across ~60 social / code / forum / creative sites. Runs concurrently — completes in ~5–15s. No API keys needed.

Per-platform deep-dives — return richer data than the People sweep. For Mastodon, use user@instance.social. For Discord, paste an invite code or full discord.gg/<code>. For YouTube, use the @handle or UC... channel ID. For PGP, search by email or fingerprint.

Wraps local CLIs — only works for binaries actually installed on this host. Hit Probe installed tools first to see which are present. All wrappers use strict input validation (no shell metachars), hard timeouts, and bounded output.
